## Installation

1.  Clone this repository or navigate to the directory.
2.  Install dependencies:

    ```bash
    npm install
    ```



### Configuration

```json
{
  "mcpServers": {
    "openskills": {
      "command": "node",
      "args": [
        "path/to/file/openskills-mcp/openskills.js"
      ]
    }
  }
}
```
